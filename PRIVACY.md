# Privacy Policy for LLM Chat Scraper

## data Collection
This extension does **not** collect, store, or transmit any user data to external servers.

## Data Usage
The extension accesses the content of the active web page (specifically Google Gemini and ChatGPT interfaces) solely for the purpose of formatting the chat history into Markdown and copying it to your local clipboard. This processing happens entirely within your browser on your local machine.

## Permissions
- **activeTab**: Used to access the content of the current tab when you activate the extension.
- **scripting**: Used to inject the content script that extracts the chat history.
- **clipboardWrite**: Used to copy the formatted text to your clipboard.
- **Host Permissions**: Limited to `https://gemini.google.com/*`, `https://chatgpt.com/*`, and `https://chat.openai.com/*` to ensuring the extension only runs on supported platforms.
